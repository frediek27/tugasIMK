Program Studi Teknik Informatika

Mempersiapkan lulusan yang kompeten di bidang teknologi informasi

Teknik Informatika

Akreditasi: A
Durasi: 4 Tahun (8 Semester)
Gelar: S.Kom (Sarjana Komputer)

Deskripsi Program

Program Studi Teknik Informatika dirancang untuk menghasilkan lulusan yang memiliki kemampuan dalam pengembangan perangkat lunak, analisis sistem, keamanan siber, dan teknologi informasi terkini. Program ini menggabungkan teori dan praktik untuk mempersiapkan mahasiswa menghadapi tantangan industri teknologi.

Mata Kuliah Unggulan

Pemrograman Web
Basis Data
Jaringan Komputer
Keamanan Siber
Machine Learning
Mobile Development

Prospek Karir

  • Software Developer/Engineer
  • System Analyst
  • Data Scientist
  • Cybersecurity Specialist
  • IT Consultant
